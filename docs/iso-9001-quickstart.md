# Guía rápida — Cómo implantar ISO 9001 en QualiQ

## 1. Configura el tenant
1. Entra en **Configuración > Configuración ISO**.
2. Activa `ISO 9001` (y opcionalmente 14001/27001/45001).
3. Define tipos documentales permitidos.
4. Ajusta mínimos de revisores y aprobadores.

## 2. Define el mapa de procesos
1. Crea procesos y subprocesos en el catálogo de procesos.
2. Asigna owners y (si aplica) KPI.
3. Relaciona documentos con procesos.

## 3. Carga y controla documentación
1. Crea procedimientos, instrucciones, formatos, políticas y registros.
2. Usa flujo: borrador → revisión → aprobado → publicado.
3. Exige resumen de cambios al crear nueva versión.

## 4. Construye la matriz de cumplimiento
1. Carga cláusulas ISO 9001 relevantes.
2. Asocia controles por cláusula.
3. Adjunta evidencias (documentos, registros, acciones).

## 5. Ejecuta auditorías internas
1. Planifica auditoría.
2. Usa checklist por cláusulas.
3. Registra hallazgos con severidad y evidencia.

## 6. Gestiona no conformidades y acciones
1. Registra NC por origen (auditoría, cliente, proceso).
2. Define contención y causa raíz.
3. Crea acciones correctivas/preventivas/de mejora.
4. Verifica eficacia y cierra con evidencia.

## 7. Mejora continua
1. Revisa riesgos y oportunidades.
2. Ejecuta seguimiento periódico de indicadores.
3. Mantén trazabilidad completa para auditoría externa.
